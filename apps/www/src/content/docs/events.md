Livedot keeps event counting simple.

## Counting rules

- One visitor session equals one event, no matter how long that visitor stays.
- Custom named events still count individually.
- Live map updates still stream every 5 seconds, so the realtime experience stays the same.

## Why this matters

Your monthly limit is based on tracked activity, not page refreshes or how long a visitor remains on the page.

That keeps plan usage predictable and makes pricing easier to understand.
